# Task Manager

Базовый FastAPI-проект для управления задачами.

## Запуск

1. Создать и активировать виртуальное окружение.
2. Установить зависимости: `pip install -r requirements.txt`.
3. Запустить приложение: `uvicorn src.app:app --reload`.

## Проверка

- Health-check: `GET /health`
- Тесты: `pytest`
